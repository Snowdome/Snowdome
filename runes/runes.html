1. 入門篇

一開始可以存50顆刻印（已裝備的刻印不計算在內），而每最大值+5盛惠100幻晶石（最大值為300）。請注意，當倉庫刻印多於最大值時，將無法開始戰鬥。（PoK玩家會好熟悉）

整個刻印系統由三種元素組成：級別、大罪屬性（形狀）和組合（符文）。級別決定基礎值，形狀決定屬性，符文決定組合加成。

目前級別分為三級：3★(藍色)、4★(黃色)及5★(紅色)。就其餘兩項元素，可以先看一看表格一得一個概念：

大罪屬性決定刻印提供的基礎屬性跟覺醒屬性的種類，而刻印的級別則決定基礎能力的範圍值，但對覺醒屬性並沒有影響。就目前的數據而言，基礎能力的範圍值並沒有重疊。因此，在不考慮成平時情況下，不存在歐洲清泉用低級刻印勝過非洲油王的情況。

除色欲及強欲外，其餘四屬均會隨機二選一出一種基礎屬性。基礎屬性可跟據加值分為三類：HP(數值大，B類）、素早（數值小，C類）、和其他（A類），各種類會對應各自的概率。概率不受刻印級別影響（即物HP 3★刷出最高級別跟5★刷出最高級的概率是一樣的）。

在刻印覺醒後，會跟據大罪屬性隨機抽出一種覺醒屬性。同樣地，覺醒屬性跟據加值分為三類：+1至+5（A類）、+1至+10（B類）、及+1至+3（C類）。每顆刻印最多可以覺醒三次，而覺醒屬性是可以重覆的。各自的加值概率分佈如下：

2. 強化篇

以上簡單介紹了刻印系統的數值部份。單看數值的確很美好（每一顆刻印都媲美甚至超過一件武具的價值）。Zahar提升了125物攻 10%物攻 820HP 15素早 15斬 5單体？！理想很美好，除了......

刻印系統的素材包括：

  1. 強化用的強化石（綠石）：關卡任務送不少的量，而且可以從刻印分解中獲得。
  2. 覺醒用的錬金鉱（碎石）：3★(藍石)、4★(黃石)及5★(紅石）。三種石都可從各自的關卡低機率刷出；藍石可以從分解3★刻印中低機率出現；黃石可以從分解4★及5★刻印中低機率出現（你無看錯，分解刻印是不會分解出紅石的。油王可以直接課金買。）
  3. 天秤：可以用作重刷覺醒的過路費
  4. 錢，很多錢
  5. 幻晶石

強化的系數分類跟表格二的基礎屬性分類是相同的（除了素早5★升滿會多比3-4★一點），而費用則是根據級別調整。

3. 實踐篇

整個刻印系統，其中最麻煩的一環便是其高昂的重刷費用。總結一下上述的概率及費用：

  • 滿覺醒的刻印需要12顆相對等級的碎石，而暫時紅石碎極其珍貴。
  • 一顆滿突的3★刻印需要最少534萬 + 63綠石，且未計算重刷覺醒屬性。
  • 滿突的3★刻印平均費用（預期值，下同）為955萬 + 173綠石。
  • 雖然綠石相對容易入手，但每一顆完成體所需的過百顆綠石並不是可以忽視的存在（至少現階段還是這樣）。
  • 同樣地，一顆滿突的4★刻印最少花費為1068萬 + 84綠石，平均費用為1830萬 + 217綠石。
  • 一顆滿突的5★刻印最少花費為1602萬 + 105綠石，平均費用為2655萬 + 257綠石。
  • 另一方面，視乎追求數值的程度，重刷開支可以出現天價（認真的）。
  • 一顆嫉妬刻印想刷到三個斬擊攻擊加成，3★平均需要750萬；4★平均需要1500幻晶石；5★平均需要3000幻晶石。
  • 如果改為目標要三個斬擊攻擊+3或以上，3★平均需要2850萬；4★平均需要5700幻晶石；5★平均需要11400幻晶石。
  • 就A組數值而言，要刷到最高階數值，3★平均需要300萬；4★平均需要600萬；5★平均需要2000幻晶石。
  • B組：3★平均需要200萬；4★平均需要400萬；5★平均需要1333幻晶石。
  • C組：3★平均需要1500萬；4★平均需要3000萬；5★平均需要10000幻晶石。

有某幾種基礎能力跟組合加成的相性相對好（粉紅色背景），例如物攻的嫉妬配合強攻。這些刻印可以多存一點無妨，方便以其作組合核心。

相反，有一些刻印與組合加成方向相反。這些（藍色背景）也值得存下，不過是存3-4★的（想要覺醒屬性便用3★；想要高基礎屬性便用4★。）

在開始前，不妨先問問自己以下問題：

  1. 六項基礎屬性分別選哪種？
  2. 今次想要哪些組合加成？
  3. 會不會有配搭方法可以使其中一些刻印於其他組合中重用？
  4. 優先基礎屬性數值（用5★），還是優先覺醒屬性（用3★）？
  5. 有沒有足夠的素材覺醒？

以下以親生仔Zahar作一個例子：
組合方面，選[4]物攻 +10%及[2]単体攻擊 +5。
基礎屬性方面，物攻優先，HP及素早為次，其他隨便。
覺醒屬性方面，要斬擊攻擊及MP回復（總數11點或以上），物攻衰弱耐性為次，其他隨便。因此，我的選擇如下：

實際操作流程：

  1. 為了減省成本，只有特殊的刻印，才會重刷基礎屬性，其餘均努力農出本體才完成第一步。目前特殊刻印只有一閃及一掃。（因為要用Draft coin換）
  2. 覺醒優先處理5★及3★。
  3. 重刷覺醒直至刷出可接受的斬擊攻擊。所須的幻晶石數量因血統而異；MP回復相對壓力少一點，只會扣錢。有餘力才重刷物攻衰弱耐性。
  4. 由於素早的強化加成效果不顯注，視乎自身財力情況可略過覺醒。同樣地，HP只屬次要屬性，視乎自身情況可略過覺醒。
  5. 完成一隻角的基本真理刻印。至少還要重複五次～

流程大致上就是這樣。最後請謹記：要量力而為，Happy farming, happy gaming!
